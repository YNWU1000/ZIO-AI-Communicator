# ZIO-AI-Communicator
ZIO is an AI communicator powered by GPT-4. It features 8 LCD1602 screens, a thermal printer, and audio input &amp; output. This DIY project uses OpenAI API, Google Cloud Platform, Raspberry Pi, and Arduino.
